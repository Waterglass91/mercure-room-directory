# Room Directory Mercure — version GitHub Pages simple

Cette version est prévue pour être modifiée directement depuis GitHub.

## Fichiers importants

- `index.html` : structure du site
- `styles.css` : design
- `script.js` : navigation et logique
- `content/hotel.json` : tous les textes modifiables
- `assets/uploads/hotel-hero.jpg` : photo principale de l'hôtel

## Pour modifier les textes

Modifier uniquement :

`content/hotel.json`

Attention : garder les guillemets, les virgules et les accolades.

## Pour modifier la photo principale

Ajouter une image dans :

`assets/uploads/`

et la nommer exactement :

`hotel-hero.jpg`

## Publication GitHub Pages

Créer un repository GitHub, envoyer ces fichiers, puis activer GitHub Pages sur la branche `main`, dossier racine `/`.

## Important

Le site doit être ouvert via GitHub Pages ou un serveur web. Si vous ouvrez `index.html` directement en double-cliquant depuis Windows, le fichier JSON peut ne pas se charger.
